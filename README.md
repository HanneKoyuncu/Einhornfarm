# Einhorn-Farm-Spiel 

## Projektbeschreibung
Es wurde ein Farmspiel entwickelt, in dem Einhörner
gesammelt, gezüchtet, verkauft und bewegt werden können.


## Funktionen
- Kauf von Einhörnern
- Kauf von Ställen
- Züchten von Einhörnern durch Attribut-Kombinationen
- Verkauf und Bewegung von Einhörnern
- Tutorial für neue Spieler

## Start des Programms
Das Spiel wird über die Main-Klasse gestartet.
